Sphinx

Installation
------------

* First we need sphinx module
        ::

            pip install sphinx

* Second we need `auto build`_ for quick preview
        :: 

            pip install sphinx-autobuild


.. _`auto build`: https://github.com/executablebooks/sphinx-autobuild
